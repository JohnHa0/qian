# Qian (wip)

> Qian is an hackable **micro File explorer** for **OSX**, written in **Elm** and
> **Electron**. The name **Qian** come from the name of
> [Zhang Qian](https://en.wikipedia.org/wiki/Zhang_Qian), a Chinese
> explorer. (aha get it ?)
